제거된 취약 요소

추가된 보안 조치 
// 보안이 강화된 코드
+ echo htmlspecialchars(
+     $_GET['input'],
+     ENT_QUOTES,
+     'UTF-8'
+ );
// HTML 인코딩 = XSS 차단

💡 학습 포인트
  • htmlspecialchars(): HTML 특수문자를 엔티티로 변환
  • ENT_QUOTES: 작은따옴표, 큰따옴표도 함께 인코딩
  • UTF-8 지정: 문자셋 공격 방지
  • 원칙: 사용자 입력은 반드시 검증/인코딩 후 사용
